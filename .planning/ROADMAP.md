Build a Tinder-style activity swiper for volunteers with drag gestures, match celebration, and seamless mock/real API switching.

### Success Criteria
- Cards can be swiped left/right with drag gestures
- Cards can be passed/matched with accessible buttons
- Match triggers celebration animation with confetti
- Mock data works when VITE_USE_MOCK_DATA=true
