<task type="auto">
  <name>Task 2: Enhance global focus states</name>
  <files>frontend/src/index.css</files>
  <action>
Add visible, consistent focus states for all interactive elements:

1. Add global focus-visible styles:
   *:focus-visible {
     outline: 3px solid var(--color-primary);
     outline-offset: 2px;
   }

2. Override for specific elements that need custom focus:
   button:focus-visible,
   a:focus-visible,
   input:focus-visible,
   select:focus-visible,
   textarea:focus-visible {
     outline: 3px solid var(--color-primary);
     outline-offset: 2px;
     border-radius: 4px;
   }

3. High contrast mode focus override:
   [data-contrast="high"] *:focus-visible {
     outline-color: #FFFF00;
   }

Focus states must be visible without needing to see the element's default styling.
Do not use outline: none anywhere except when replacing with equivalent visible indicator.
  </action>
  <verify>Tab through page shows clear coral outline on all focusable elements</verify>
  <done>All buttons, links, inputs show 3px coral outline when focused via keyboard</done>
</task>

1. In index.css, add OpenDyslexic font import (use CDN):

   - Add dyslexicFont state (boolean, persisted to localStorage)
   - Add setDyslexicFont function
   - In useEffect, set data-font attribute:
     document.documentElement.setAttribute('data-font', dyslexicFont ? 'dyslexic' : 'default')

3. In AccessibilityToolbar.jsx:
   - Add dyslexic font toggle button (use BookOpen icon from lucide-react)
   - Label: "Readable font" 
   - Shows active state when dyslexicFont is true

4. Update CSS for font switching:
   [data-font="dyslexic"] body,
   [data-font="dyslexic"] * {
     font-family: 'OpenDyslexic', sans-serif !important;
   }
  </action>
